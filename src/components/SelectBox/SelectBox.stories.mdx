# SelectBox

The `SelectBox` component is used for rendering customizable select boxes. It provides various props to configure its appearance and behavior.

## Examples

### Default

You can use the `SelectBox` component with default settings. Here's how to use some of its key properties:

- `className`: Add a custom class name to the component.
- `name`: The name attribute for the select input.
- `label`: The label to display for the select box.
- `value`: Specify the currently selected value(s).
- `allowCreate`: Allow users to create new options.
- `max`: Set the maximum number of selections allowed for multi-select.
- `multi`: Enable multi-select.
- `hasSearch`: Enable or disable the search functionality.
- `placeholder`: Specify a custom placeholder text.
- `options`: Provide an array of options, each with a `title` and `value`.
- `onChange`: Handle changes in the selected value(s).

The `className` prop allows you to add a custom class name to the component. You can use it to apply custom CSS styles to the select box.

  </Story>

### Custom name

The name prop specifies the name attribute for the select input. It is used to identify the select box in form submissions.

The label prop defines the label to display for the select box. It provides context to users about the purpose of the select box.

<Canvas>

The value prop allows you to specify the currently selected value(s) in the select box. It accepts an array of strings.

### List item creation

The allowCreate prop, when set to true, allows users to create new options in the select box.

<Canvas>
  <Story
    name="Allow Create"

### Maximum allowed selections

The max prop sets the maximum number of selections allowed for multi-select. It limits the number of options that can be chosen.

<Canvas>
  <Story
    name="Maximum Selections (Multi-select)"
    args={{

      label: 'Choose Pokemon',

  >
    {Template.bind({})}
  </Story>

The hasSearch prop, when set to true, enables the search functionality in the select box, allowing users to filter options.

<Canvas>

### Placeholder

The placeholder prop specifies a custom placeholder text to display when no option is selected. If not provided, the label is used as the placeholder.

<Canvas>
  <Story
    name="Custom Placeholder"
    args={{
      placeholder: 'Select a Pokemon', // Set a custom placeholder
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

### Custom Options

The options prop is an array of options, each with a title and value. It populates the select box with available choices.

<Canvas>
  <Story
    name="Options"

### on Change callback functionality

The onChange prop is a callback function that handles changes in the selected value(s). It receives an object with name and value properties.

<Canvas>
