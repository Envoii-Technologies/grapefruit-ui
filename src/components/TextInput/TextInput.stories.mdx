# TextInput

The `TextInput` component is used for rendering customizable input fields. It supports various props to configure its appearance and behavior.

## Examples

### Default

You can use the `TextInput` component with default settings. Here's how to use some of its key properties:

- `label`: The label to display for the input field.
- `type`: The type of the input field (e.g., 'text', 'password').
- `name`: The name attribute for the input field.
- `placeholder`: The placeholder text for the input field.
- `metaLabel`: An optional label for additional metadata.
- `metaPosition`: The position of the metadata label (either 'left' or 'right').


<Canvas>
  <Story
    name="Default"
    args={{
      label: 'Username',
      type: 'text',
      name: 'username',
      placeholder: 'Enter your username',
    }}
  >

</Canvas>
