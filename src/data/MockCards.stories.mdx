# Cards

The following is a set of mock data, to emulate the supplied data for a card collection.

## Data

The following data array is randomly created,
each card supplies the following informations:

- title: The title of the card
- description: The description of the card
- released: is the card released?
- deleted: is the card marked for deletion?
- version: version of the card, derived from database changes
- createdBy: Card created by user
- updatedBy: Card updated by user
- createdAt: Card creation timestmap
- updatedAt: Card updated timestamp
